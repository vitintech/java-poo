### **Aplicando na prática - POO**



###### Esse arquivo tem como objetivo aplicar POO, de forma básica em jogo de adivinhação



###### **Requisitos:**

* O sistema precisa gerar um número aleatório de 1 a 100
* O usuário precisa ter 5 tentativas, toda vez que iniciar o jogo
* A cada tentativa errada, o sistema deve diminuir as tentativas restantes e falar se o numero que ele digitou e maior ou menor ao numero sorteado
* Se o usuário acertar, deve aparecer uma mensagem de parabéns
* Se acabar as tentativas do usuário e ele não tiver acertado, ele deve reiniciar o jogo



###### **Funcionalidades:**

* Para gerar um número aleatório em Java: new Random().nextInt(100);
* Utilize o Scanner para obter os dados do usuário;
* Utilize uma variável para contar as tentativas;
* Utilize um loop para controlar as tentativas;
* Utilize a instrução break; para interromper o loop
