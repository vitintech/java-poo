### **🎮 Aplicando na Prática — Programação Orientada a Objetos (POO)**



##### **📌 Objetivo**



###### Este projeto tem como objetivo aplicar conceitos básicos de Programação Orientada a Objetos (POO) através do desenvolvimento de um jogo de adivinhação em Java.

###### 

###### A proposta é utilizar estruturas fundamentais da linguagem para criar uma aplicação simples, mas funcional, que permita ao usuário tentar adivinhar um número gerado aleatoriamente pelo sistema.



##### **⚙️ Requisitos do Sistema**



###### **O jogo deve seguir as seguintes regras:**



* O sistema deve gerar um número aleatório entre 1 e 100.
* O usuário terá 5 tentativas para acertar o número.



A cada tentativa:



* O sistema deve informar se o número digitado é maior ou menor que o número sorteado.
* O número de tentativas restantes deve ser reduzido.
* Se o usuário acertar o número, deve aparecer uma mensagem de parabéns.
* Caso o usuário não acerte dentro das 5 tentativas, o jogo deve finalizar e mostrar o número secreto



##### **🛠️ Funcionalidades Utilizadas**



###### **Para implementar o jogo, serão utilizados os seguintes recursos da linguagem Java:**



* Geração de número aleatório: new Random().nextInt(100);
* Entrada de dados do usuário, utilizando a classe Scanner para capturar valores digitados pelo usuário.



###### **Controle de tentativas**



* Uma variável será utilizada para contar e limitar o número de tentativas.



###### **Estrutura de repetição**



* Um loop será utilizado para controlar o fluxo do jogo, sendo o laço de repetição do while.
* Interrupção do loop, a instrução break será utilizada para encerrar o loop quando o usuário acertar o número.



##### **🎯 Conceitos Aplicados**



###### **Este exercício permite praticar:**



* Estruturas de decisão (if / else)
* Estruturas de repetição (while)
* Entrada de dados com Scanner
* Geração de números aleatórios com Random
* Lógica de controle de tentativas
* Organização básica de um programa em Java
